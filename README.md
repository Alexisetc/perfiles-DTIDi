## 🧭 Cómo usar

1. Doble clic en `Funcionario DTIDi-Guayaquil.html`.
2. Navegar las tarjetas agrupadas en 5 clusters temáticos.
3. Click en una tarjeta → modal con 3 pestañas:
   - **Perfil Académico** – trayectoria, proyectos, publicaciones.
   - **Sinergia EpiSIG** – propuesta de colaboración concreta para la reunión.
   - **Fuentes** 🔗 – lista numerada de URLs públicas verificables (ResearchGate, PubMed, ORCID, sitio oficial INSPI, INSPILIP, MSP, etc.).
